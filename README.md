# ⏱ Dark Stopwatch Web Application

An interactive and responsive stopwatch web application built using HTML, CSS, and JavaScript.

## Features

- Start stopwatch
- Pause stopwatch
- Reset stopwatch
- Record lap times
- Millisecond precision
- Dark modern UI
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

stopwatch-project/
│
├── index.html
├── style.css
├── script.js
└── README.md

## Run Locally

1. Download or clone the repository
2. Open `index.html` in your browser
